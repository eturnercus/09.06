# WatchAlert Tab — расширение для браузера

Мониторинг **зон на вкладках** со звуковым сигналом, если изменение сохраняется заданное время.  
Можно переключаться на другие вкладки — наблюдение за закреплёнными вкладками идёт **в фоне**.

Работает в **Chrome** и **Edge** (Chromium, Manifest V3).

## Установка (режим разработчика)

1. Откройте `chrome://extensions/`
2. Включите **«Режим разработчика»**
3. **«Загрузить распакованное расширение»** → выберите папку `extension/`

## Как пользоваться

1. Откройте нужную вкладку (сайт, дашборд, стрим и т.д.)
2. Нажмите на иконку WatchAlert → **«+ Текущая»**
3. **«+ Зона»** — выделите прямоугольник на странице
4. Повторите для других зон или добавьте другие вкладки
5. **«▶ Старт»** — мониторинг запущен
6. Переключайтесь на другие вкладки — наблюдаемые остаются активными (`autoDiscardable: false`)

## Настройки

| Параметр | Описание |
|----------|----------|
| Задержка | Секунды до сигнала при устойчивом изменении |
| Чувствительность | Порог отличия кадров (меньше = чувствительнее) |
| Звук | Свой аудиофайл или встроенный сигнал |

## Как это работает

- Зоны задаются в **долях окна вкладки** (0–1), привязаны к видимой области
- Фоновый **offscreen**-документ захватывает кадры через `tabCapture` — вкладка не обязана быть на экране
- Логика сигнала та же, что в десктопном WatchAlert: одно срабатывание на изменение, без повтора без нового отличия

## Ограничения

- Не работает на `chrome://`, Web Store и некоторых системных страницах
- На страницах с жёстким CSP выбор зоны может не сработать
- Захват вкладки требует разрешений расширения; после перезапуска браузера нажмите **Старт** снова
- Видео/тяжёлые страницы нагружают CPU при частом опросе

## Связь с десктопным WatchAlert

| | Десктоп | Расширение |
|---|---------|------------|
| Область | Любой экран | Вкладка браузера |
| Фон | Всегда | Другие вкладки OK |
| ОС | Linux/Windows | Chrome/Edge |
