# WatchAlert Tab — расширение для браузера

Мониторинг **зон на вкладках** со звуковым сигналом, если изменение сохраняется заданное время.  
Можно переключаться на другие вкладки — наблюдение за закреплёнными вкладками идёт **в фоне**.

Поддерживаются **Chrome / Edge** (Chromium) и **Firefox**.

## Установка из релиза

На [странице Releases](https://github.com/eturnercus/09.06/releases) скачайте:

| Файл | Браузер |
|------|---------|
| `WatchAlert-Tab-chrome.zip` | Chrome, Edge, Brave и др. Chromium |
| `WatchAlert-Tab-firefox.zip` | Firefox 121+ |

Распакуйте zip в любую папку и установите распакованное расширение (см. ниже).

Локальная сборка zip:

```bash
./build/build-extension.sh
```

## Установка (режим разработчика)

### Chrome / Edge

1. Распакуйте `WatchAlert-Tab-chrome.zip` **или** используйте папку `extension/` из репозитория
2. Откройте `chrome://extensions/`
3. Включите **«Режим разработчика»**
4. **«Загрузить распакованное расширение»** → выберите папку

### Firefox

1. Распакуйте `WatchAlert-Tab-firefox.zip`
2. Откройте `about:debugging#/runtime/this-firefox`
3. **«Загрузить временное дополнение…»** → выберите `manifest.json` из распакованной папки

Для постоянной установки без режима разработчика нужна подпись в AMO (пока — только временная загрузка или распакованная папка через `about:config` `xpinstall.signatures.required=false` на свой риск).

## Как пользоваться

1. Откройте окно с нужными вкладками (дашборд, стрим и т.д.)
2. (Опционально) **«Закрепить текущее окно»** — мониторинг только в этом окне; при **Старт** расширение найдёт его по URL вкладок, даже после перезапуска браузера
3. **«+ Текущая»** — добавить вкладку в список
4. **«+ Зона»** — выделить прямоугольник на странице
5. **«▶ Старт»** — мониторинг запущен; можно работать в других окнах
6. **«■ Стоп»** — остановить

Если окно закреплено, вкладки добавляются только из него.

## Настройки

| Параметр | Описание |
|----------|----------|
| Задержка | Секунды до сигнала при устойчивом изменении |
| Чувствительность | Порог отличия кадров (меньше = чувствительнее) |
| Звук | Кнопка «Выбрать файл…» (отдельная вкладка) или встроенный сигнал |

## Как это работает

- Зоны задаются в **долях окна вкладки** (0–1), привязаны к видимой области
- **Chrome:** offscreen-документ + `tabCapture` — захват вкладки в фоне
- **Firefox:** `tabs.captureTab()` в service worker (API Firefox, фоновые вкладки)
- Логика сигнала та же, что в десктопном WatchAlert: одно срабатывание на изменение, без повтора без нового отличия

## Ограничения

- Не работает на `chrome://`, `about:`, Web Store и некоторых системных страницах
- На страницах с жёстким CSP выбор зоны может не сработать
- После перезапуска браузера нажмите **Старт** снова
- **Firefox:** закреплённая служебная вкладка для захвата и звука (закрывается при **Стоп**)
- Видео/тяжёлые страницы нагружают CPU при частом опросе
- Звуковой файл до 8 МБ

## Связь с десктопным WatchAlert

| | Десктоп | Расширение |
|---|---------|------------|
| Область | Любой экран | Вкладка браузера |
| Фон | Всегда | Другие вкладки OK |
| ОС | Linux/Windows | Chrome/Edge/Firefox |
